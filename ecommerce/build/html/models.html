Modèles (Models)

Le projet contient plusieurs modèles pour gérer les événements, les formules, les utilisateurs, et les commandes.

Modèle Discipline

Ce modèle représente les différentes disciplines sportives des JO 2024.

  • name : nom de la discipline.

  • description : description de la discipline.

  • image : image associée.

Modèle Formule

Ce modèle représente les formules disponibles pour chaque événement.

  • formule : nom de la formule.

  • price_multiplier : multiplicateur de prix appliqué à l’événement.
